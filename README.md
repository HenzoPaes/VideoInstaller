# 🎬 YouTube Video Downloader 

Aplicativo simples e intuitivo para baixar vídeos e áudios do YouTube, com suporte a múltiplos idiomas e FFmpeg integrado.

---

## 🚀 Funcionalidades

- 📥 Download de vídeos em MP4
- 🎵 Download de áudio (MP3, M4A, etc)
- 🌍 Seleção de idioma do áudio
- 📂 Abre automaticamente a pasta do arquivo baixado
- ⚙️ Painel avançado (FFmpeg, logs)
- 🔄 Verificação automática de atualizações

---

## 🖥️ Requisitos

- **Windows**
- **Python** 

> O executável final **não exige Python instalado - eu acho**.
